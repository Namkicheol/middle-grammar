# 기초영문법 블로그 글 작성 가이드

> 티스토리 블로그 **기초영문법** 카테고리 전용 작성 규칙

---

## 카테고리 현황

- **글 번호**: 15(be동사기초), 16(동명사기초), 17(동명사심화), 31(일반동사심화), 32(be과거기초), 33(be과거심화), 34~35(to부정사기초/심화)
- **형식**: 마크다운
- **톤**: 친근함 (중학생 대상)
- **색깔**: 파랑(문법 용어) · 주황(팁) · 빨강(함정) 3색 중심, 보라 거의 안 씀

---

## 색깔 시스템

티스토리 마크다운 에디터는 HTML `<span>` 태그 허용 → 인라인 색상 적용 가능

| 용도 | 색상 코드 | 패턴 예시 |
|---|---|---|
| 핵심 개념·정의 | `#3182ce` (파랑) | `<span style="color:#3182ce;">**개념어**</span>` |
| 주의·경고·함정·비문 | `#c53030` (빨강) | `<span style="color:#c53030;">**주의**</span>` |
| 현직쌤 팁 박스 | `#dd6b20` (주황) | `> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>:` |
| 장점·긍정·✅ | `#319795` (청록) | `<span style="color:#319795;">장점</span>` |

### 색깔 사용량 가이드

| 글 길이 | span 권장 개수 |
|---|---|
| 5,000~8,000자 | 6~10개 |
| 8,000~12,000자 | 10~15개 |

### 색깔 금기 사항

- 한 문단에 4가지 색 이상 ❌
- 전체 문장을 색으로 감싸지 말 것 (키워드 1~3단어만)
- 헤더(`##`, `###`)에 색 입히지 말 것
- 표(table) 안에 색 최소화
- 중복 강조 금지 (볼드+색 된 단어에 밑줄·이탤릭 추가 ❌)

## SVG 도해 가이드

### 만들어야 하는 경우

- 위계 구조, 과정·흐름, 비교·대조, 분류 체계

### 불필요한 경우

- 단순 정의 나열 → 표로 대체
- 짧은 대비(2~3항목) → 표 2열로 대체

### 표준 템플릿

```html
<p align="center">
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 700 280" width="700" style="max-width:100%;height:auto;">
  <rect x="0" y="0" width="700" height="280" fill="#fafafa" stroke="#e2e8f0"/>
  <text x="350" y="28" text-anchor="middle" font-size="15" font-weight="bold" fill="#2d3748">제목</text>
  <!-- 내용 -->
</svg>
</p>
```

### SVG 색상 팔레트

| 박스 유형 | 배경색 | 테두리색 |
|---|---|---|
| 파랑 | `#ebf8ff` | `#3182ce` |
| 청록 | `#e6fffa` | `#319795` |
| 주황 | `#fef5e7` | `#dd6b20` |
| 빨강 | `#fff5f5` | `#c53030` |

- 기본 가로: `viewBox="0 0 700 ..."`
- 글씨: 제목 15~16px, 본문 10~12px, 주석 9px

## 이미지 삽입 가이드

- **출처**: Pexels만 사용
- **크기**: `w=600`, 3~5장
- **URL 패턴**: `https://images.pexels.com/photos/{id}/pexels-photo-{id}.jpeg?w=600`
- **허구 ID 절대 생성 금지** — 실제 존재하는 URL만 사용

```html
<p align="center">
<img src="https://images.pexels.com/photos/XXXX/pexels-photo-XXXX.jpeg?w=600" alt="설명" width="600"/>
</p>
```

---

## 글 작성 공통 규칙

- **~ 물결표**: 앞뒤 공백 필수 → `A ~ B` (티스토리 취소선 오류 방지)
- **색깔 시스템**: 글 작성 시 디폴트 적용. "색깔 빼줘" 요청 시에만 제외
- **SVG**: 위계·흐름·비교 구조 있으면 제안 없이 바로 제작
