## 블로그 본문

수동태, 다들 이해는 했는데 막상 변환하면 헷갈리지 않아?

"Jake broke the window."를 수동태로 바꾸라고 하면 "The window was broken..."까지는 쓰는데, 뒤에 **by Jake**를 붙여야 한다는 걸 깜빡하거나, 아니면 반대로 수동태를 능동태로 바꾸려면 뭘 어디에 써야 할지 멈칫하는 경우 많잖아.

오늘 소개하는 워크시트는 **능동 → 수동**, **수동 → 능동** 양방향 변환만 집중 훈련하는 20문제짜리 변환 특훈이야.

## 수동태 변환, 이게 핵심이야!

수동태 변환의 핵심은 **주어와 목적어의 자리 교환**이야. 색깔로 기억하면 훨씬 쉬워.

| | <span style="color:#dd6b20;">주어(행위자)</span> | <span style="color:#3182ce;">동사</span> | <span style="color:#319795;">목적어(대상)</span> | |
|:--:|:--:|:--:|:--:|:--:|
| **능동태** | <span style="color:#dd6b20;">**Jake**</span> | <span style="color:#3182ce;">**broke**</span> | <span style="color:#319795;">**the window**</span> | |
| | ↕ | ↕ | ↕ | 같은 색끼리 이동! |
| **수동태** | | <span style="color:#3182ce;">**was broken**</span> | <span style="color:#319795;">**The window**</span>→주어 | <span style="color:#dd6b20;">**by Jake**</span> |

> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>: 능동태 **목적어**(초록)가 수동태에서 **주어**가 되고, 능동태 **주어**(주황)는 **by 뒤**로 가. 이 두 가지만 기억하면 변환 끝!

## 변환할 때 이것만 챙겨!

**<span style="color:#c53030;">능동 → 수동 (핵심 3단계)</span>**

① 목적어 → 주어 자리로 이동
② 동사 → be동사 + p.p. (단수→was/is · 복수→were/are)
③ 원래 주어 → by + 주어 *(by는 생략 가능, 시험엔 주로 포함)*

**<span style="color:#319795;">수동 → 능동 (핵심 3단계)</span>**

① by 뒤 명사 → 주어 자리로 이동
② be + p.p. → 능동태 동사 (과거수동→과거형 · 현재수동→현재형)
③ 수동태 주어 → 목적어 자리로 이동 *(3인칭 단수 주어면 -s/-es 주의!)*

> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>: 수동 → 능동 변환에서 제일 많이 틀리는 게 **3인칭 단수 -s**야. "is made by Mia's mom" → "Mia's mom **makes**" — 현재형이면 3인칭 단수 주어에 -s를 붙이는 거 잊지 마!

| 탭 | 내용 | 문제 수 |
|----|------|---------|
| 🔄 능동 → 수동 | 능동태 문장을 수동태(be + p.p.)로 변환 | 10문제 |
| 🔃 수동 → 능동 | 수동태 문장을 능동태(동사형)로 변환 | 10문제 |

빈칸에 be동사+p.p. 또는 능동 동사형을 직접 써야 해서, 단순히 고르는 것보다 **훨씬 실력이 잘 늘어**! 틀리면 바로 힌트와 해석이 뜨니까 이 워크시트 한 번으로 수동태 변환 완전 정복!
