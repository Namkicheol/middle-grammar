## so ~ that, 이게 뭐야?

<span style="color:#3182ce;">**so ~ that**</span> 구문은 "너무 ~해서 …하다"라는 뜻이야.
**원인**과 **결과**를 한 문장에 우아하게 연결하는 표현이지.

> 예: It was **so** cold **that** I wore a coat.
> (너무 추워서 나는 코트를 입었다.)

| 구성 | 역할 | 예시 |
|:----:|:----:|:----:|
| 주어 + 동사 | 원인 시작 | It was |
| **so** | 강조 부사 | so |
| 형용사/부사 | 상태·정도 | cold |
| **that** | 결과 접속사 | that |
| 결과절 | 결과 | I wore a coat. |

← 원인 부분 (주어 + so + 형용사/부사) &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 결과 (that + 절) →

> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>: `very`와 `too`는 that절을 이끌 수 없어. "It was **very** cold **that** …" 이런 문장은 틀린 거야. so만 가능해!

<span style="color:#c53030;">**주의**</span> — **형용사 vs 부사** 함정도 조심해:

| 상황 | 쓸 것 | 예시 |
|------|-------|------|
| be동사·감각동사 뒤 | **형용사** | so **cold** / so **good** |
| 일반동사(행동) 뒤 | **부사** | so **quickly** / so **beautifully** |

`She sang so beautiful.` → <span style="color:#c53030;">**틀림**</span>  
`She sang so beautifully.` → ✅ 맞음

---

## 수동태, 이렇게 만들어!

<span style="color:#3182ce;">**수동태**</span>는 "~되다 / ~당하다"처럼 주어가 동작을 **받는** 표현이야.
만드는 공식은 딱 하나야:

<p align="center"><span style="font-size:20px;font-weight:bold;color:#2b6cb0;background:#ebf8ff;padding:10px 24px;border-radius:10px;border:2px solid #3182ce;display:inline-block;">수동태 = be동사 + p.p.</span></p>

**be동사는 주어의 수와 시제에 맞춰 골라야 해:**

| 시제 | 단수 주어 | 복수 주어 |
|:----:|:--------:|:--------:|
| 현재 | **is** | **are** |
| 과거 | **was** | **were** |

**능동 → 수동, 이렇게 바뀌어:**

> **능동**: Jake **broke** the window.
> **수동**: The window **was broken** by Jake.
>
> 목적어(the window) → 주어 / 동사 → be + p.p. / 주어(Jake) → by + 행위자

> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>: `is making`(능동 진행형)과 `is made`(수동태)는 완전히 달라. 수동태는 반드시 **be + p.p.** 형태!

## 블로그 본문 (심화)

기초를 다졌다면 이제 **심화**로 넘어올 차례야!

심화 워크시트는 단순 고르기를 넘어서 **오류 찾기 · 문장 바꾸기 · 우리말 영작 · 짧은 지문 읽기**까지 다뤄.
실제 학교 시험에 나오는 유형과 거의 똑같거든.

---

## 심화에서 자주 틀리는 패턴 4가지

| 오류 유형 | <span style="color:#c53030;">✗ 틀린 문장</span> | ✓ 맞는 문장 | 핵심 |
|:--------:|------------|------------|------|
| **① so 자리** | It was **very** cold that I stayed home. | It was **so** cold that I stayed home. | very·too → that절 불가, so만 가능 |
| **② that 자리** | She was so tired **which** she slept. | She was so tired **that** she slept. | which·because 불가, that만 가능 |
| **③ 수동태 형태** | The bread **is making** with cream. | The bread **is made** with cream. | is making(진행) ≠ is made(수동) |
| **④ was/were** | The cakes **was** baked by Mia. | The cakes **were** baked by Mia. | 복수→were · 단수→was |

> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>: 수동태에서 가장 많이 틀리는 건 **수일치(is/are/was/were)**야. 주어가 복수인지 단수인지, 시제가 현재인지 과거인지 두 가지를 동시에 체크하는 습관을 만들어야 해.
